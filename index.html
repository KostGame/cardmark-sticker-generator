Что такое CardMark

CardMark — это маленькая векторная метка, которую можно распознать по геометрии без ИИ, OCR, внешних API и серверной обработки.

Как клеить

  1. Разложите колоду по порядку.
  2. Выберите нужный пресет.
  3. Распечатайте лист стикеров.
  4. Берите наклейки по порядку.
  5. Клейте метку в верхний левый угол лицевой стороны карты.
  6. Стрелка «верх» должна смотреть к верхнему краю карты.
  7. После наклейки отметьте чекбокс на листе.
  8. Для Таро лучше использовать протекторы, чтобы не портить карты.

Почему текст на подложке

После отделения наклейки на карте остаются только метка и маленький ориентир верха. Название, ID, группа и чекбокс остаются на бумажной подложке как рабочая инструкция.

JSON-манифест

JSON связывает ID метки с названием карты. Будущее приложение распознавания сможет найти CardMark на фото, определить ID и сопоставить его с этим манифестом.
